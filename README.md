# Quiz-app

## Build Setup

``` bash
# with Docker (recommended)
$ docker-compose up

# or build locally
$ yarn install
$ yarn run build
# make sure you have MongoDB running first and URI string in ./server/database/index.js points to localhost
$ yarn start

```
